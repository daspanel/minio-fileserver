Usage

Get

$ docker pull daspanel/minio-fileserver:latest

Run

$ docker run -e DASPANEL_MASTER_EMAIL=my@email.com --name=my-minio-fileserver daspanel/minio-fileserver:latest

Stop

$ docker stop my-minio-fileserver

Update image

$ docker stop my-minio-fileserver
$ docker pull daspanel/minio-fileserver:latest
$ docker run -e DASPANEL_MASTER_EMAIL=my@email.com --name=my-minio-fileserver daspanel/minio-fileserver:latest
